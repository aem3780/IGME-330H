API

  • API home- https://openweathermap.org/
  • API documentation- https://openweathermap.org/current
  • API city end point format- api.openweathermap.org/data/2.5/weather?q={city name}&appid={API key}
  • API zip code end point format- api.openweathermap.org/data/2.5/weather?zip={zip code},{country code}&appid={API key}
  • API key is required

Noteworthy

  • I used Bulma, web components, await fetch in JavaScript, and an API
  • My application is mobile friendly and adapts to various screen sizes
  • All JS code is housed in seperate ES6 modularized files
  • Random weather fact is displayed when page loads
  • Carousel used to show images on About page
