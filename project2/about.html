About Weather Watch

Weather Watch is a web application that utilizes the Current Weather API from OpenWeatherMap. Users are able to search for the current weather of a location by selecting from a list of cities in the US, searching by zip code, or selecting from a list of international cities. Feel free to save your favorite locations and zip codes to the favorites page for easier access to your favorite locations' weather conditions. This application uses Bulma, Mapbox, JavaScript, HTML, CSS and also local storage. Local storage allows the users search terms and saved locations to be available upon reopening of the app and Mapbox shows selected locations in map view. Have fun!
